# Zara Challenge v1

Pequeña aplicación SPA desarrollada con React y Tailwind como prueba técnica.

## 🚀 Tecnologías

* React
* Tailwind CSS
* Vite

## ✅ Estado actual

* Estructura base del proyecto
* Componentes principales
* Primeros estilos aplicados

## 🚀 Run with Docker

### Development
docker compose up

### Production
docker build -t zara-challenge .
docker run -p 3000:3000 zara-challenge
